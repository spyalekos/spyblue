# SpyBlue - Γέφυρα Ήχου από Android σε Mac

Το **SpyBlue** είναι μια εφαρμογή σε Python που μετατρέπει το Mac σας σε ασύρματο ηχείο για τη συσκευή σας Android. Χρησιμοποιεί το πρωτόκολλο **sndcpy** για τη λήψη ψηφιακού ήχου PCM υψηλής ποιότητας (16-bit stereo, 48kHz) μέσω ADB (Android Debug Bridge), είτε μέσω καλωδίου USB είτε εντελώς ασύρματα μέσω Wi-Fi.

---

## 🌟 Χαρακτηριστικά

*   **Ασύρματη Μετάδοση Ήχου (Wi-Fi)**: Μετά την πρώτη σύνδεση μέσω USB, η εφαρμογή ενεργοποιεί αυτόματα το ασύρματο debugging στο κινητό, επιτρέποντάς σας να αποσυνδέσετε το καλώδιο και να ακούτε μουσική ασύρματα.
*   **Αυτόματη Επανασύνδεση**: Αποθηκεύει την IP διεύθυνση της συσκευής σας, έτσι ώστε σε επόμενες εκτελέσεις να συνδέεται αυτόματα ασύρματα χωρίς να χρειάζεται καθόλου καλώδιο USB.
*   **Αυτόματη Σίγαση Κινητού**: Όταν συνδέεται η εφαρμογή, το ηχείο του κινητού σιγάζει αυτόματα (Mute) και ο ήχος ακούγεται μόνο στο Mac. Κατά την έξοδο, η ένταση του κινητού επανέρχεται στο αρχικό της επίπεδο.
*   **Σύγχρονο Live Dashboard**: Εμφανίζει σε πραγματικό χρόνο την κατάσταση σύνδεσης, τον ρυθμό μεταφοράς δεδομένων (KB/s), την ποιότητα της προσωρινής μνήμης (buffer queue health) και ένα ζωντανό visualizer της έντασης του ήχου.
*   **Έξοδος με Διπλό ESC**: Μπορείτε να κλείσετε την εφαρμογή ανά πάσα στιγμή πατώντας το πλήκτρο **`Escape` δύο φορές γρήγορα** (ή με `Ctrl + C`).

---

## 🛠️ Προαπαιτούμενα

1.  **Android 10 ή νεότερο** στη συσκευή σας.
2.  **Ενεργοποιημένο το USB Debugging** στο κινητό σας:
    *   Μεταβείτε στις *Ρυθμίσεις > Σχετικά με το τηλέφωνο*.
    *   Πατήστε το *Αριθμός κατασκευής (Build Number)* 7 φορές για να ενεργοποιήσετε τις *Επιλογές προγραμματιστή*.
    *   Μεταβείτε στις *Επιλογές προγραμματιστή* και ενεργοποιήστε το **Ενεργοποίηση USB (USB Debugging)**.
3.  Mac και Android συσκευή πρέπει να είναι συνδεδεμένα στο **ίδιο δίκτυο Wi-Fi** (για την ασύρματη λειτουργία).
4.  Εγκατεστημένο το Android SDK (Platform Tools / `adb`) στο Mac σας (ανιχνεύεται αυτόματα στην προεπιλεγμένη διαδρομή `~/Library/Android/sdk/platform-tools/adb`).

---

## 🚀 Οδηγίες Χρήσης

### 1. Πρώτη Εκτέλεση (Μέσω USB)

1.  Συνδέστε το κινητό σας στο Mac με ένα **καλώδιο USB**.
2.  Τρέξτε το εκτελέσιμο αρχείο `spyblue` (που βρίσκεται στο φάκελο `dist` ή μπορείτε να το κατεβάσετε από τα Releases) από το τερματικό σας:
    ```bash
    ./spyblue
    ```
    *(Εναλλακτικά, αν τρέχετε από τον πηγαίο κώδικα, χρησιμοποιήστε: `uv run main.py`)*
3.  **Στο Android**: Θα εμφανιστεί ένα παράθυρο επιβεβαίωσης. Επιτρέψτε το USB debugging από τον υπολογιστή σας.
4.  Η εφαρμογή θα κατεβάσει αυτόματα το `sndcpy.apk`, θα το εγκαταστήσει στο κινητό και θα ζητήσει άδεια καταγραφής ήχου.
5.  **Στο Android**: Πατήστε **Έναρξη τώρα (Start now)** στο αναδυόμενο παράθυρο καταγραφής ήχου.
6.  Μόλις δείτε στο τερματικό το μήνυμα **`Wireless Enabled! Unplug USB.`**, μπορείτε να **αποσυνδέσετε το καλώδιο USB**. Ο ήχος θα συνεχίσει να μεταδίδεται ασύρματα!

---

### 2. Επόμενες Εκτελέσεις (Εντελώς Ασύρματα)

Εφόσον το κινητό σας και το Mac είναι στο ίδιο Wi-Fi, δεν χρειάζεται να ξαναχρησιμοποιήσετε καλώδιο:
1.  Ανοίξτε το τερματικό και τρέξτε:
    ```bash
    ./spyblue
    ```
    *(Εναλλακτικά, αν τρέχετε από τον πηγαίο κώδικα: `uv run main.py`)*
2.  Η εφαρμογή θα διαβάσει την αποθηκευμένη IP και θα συνδεθεί αυτόματα ασύρματα.
3.  Πατήστε **Έναρξη τώρα (Start now)** στην οθόνη του κινητού σας όταν σας ζητηθεί.

---

## 🛑 Τερματισμός Εφαρμογής

Πατήστε το πλήκτρο **`Escape` δύο φορές γρήγορα** (ή `Ctrl + C`) στο παράθυρο του τερματικού. 
Η εφαρμογή θα:
1.  Διακόψει την ασύρματη μετάδοση.
2.  Επαναφέρει την ένταση του ήχου στο κινητό σας στο επίπεδο που βρισκόταν πριν τη σύνδεση.
3.  Αποδεσμεύσει τις θύρες δικτύου του Mac.
